Product Variant Measure

Este módulo extiende la gestión de variantes de producto permitiendo definir longitudes, anchos u otras magnitudes numéricas como valores de atributos, valiéndose de las unidades de medida estándar de Odoo.

Características principales

  • Permite asociar una categoría de unidad de medida a cada atributo.
  • Cada valor de atributo incluye la unidad de medida correspondiente.
  • Al ingresar solo la magnitud numérica, el sistema completa automáticamente el nombre (por ejemplo, “10” ⇒ “10 ft”).
  • Valida que los valores respeten la unidad, magnitud positiva y redondeo configurado.
  • Muestra la unidad tanto en formularios como en listas de variantes de producto.

Casos de uso

  • Gestionar variantes por largo/ancho en muebles o piezas de construcción.
  • Controlar atributos dimensionales de textiles, cables o perfiles metálicos.
  • Crear variantes con medidas expresadas en pies, metros u otras unidades de Odoo.

Cómo empezar

  1. Instale el módulo y abra Inventario ▸ Configuración ▸ Atributos.
  2. Seleccione un atributo y defina su categoría de unidad.
  3. Agregue valores indicando solo la magnitud numérica y elija la unidad; el nombre se autocompleta.
  4. Asocie el atributo a las plantillas de producto para generar variantes según las medidas configuradas.

Notas

  • Durante la instalación se eliminan las restricciones originales que impedían repetir una unidad dentro del mismo atributo.
  • Si necesita desactivar la validación temporalmente, puede usar el contexto skip_variant_measure_validation en acciones específicas.
