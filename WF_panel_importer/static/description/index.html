WF Panel Importer

Este módulo automatiza la captura de paneles tipo woodframe a partir de planos PDF del proveedor. El pipeline combina OCR y análisis geométrico para detectar cada panel, interpretar sus capas y generar en Odoo los registros maestros listos para fabricar: paneles, listas de materiales, rutas y órdenes.

Flujo principal

  1. Cargar en Odoo el PDF exportado desde la herramienta CAD compatible.
  2. Ejecutar el asistente de importación para procesar los planos.
  3. Revisar el informe de diagnóstico generado (paneles detectados, advertencias y descartes).
  4. Confirmar los paneles importados y planificar la producción usando las BOM y MO creadas.

Requisitos de configuración

  • Definir el parámetro del sistema wf_panel_importer.output_dir en Ajustes > Parámetros del sistema. Indica la carpeta donde se guardan los PDFs procesados, los archivos intermedios y los reportes de la importación. Se crea automáticamente con el valor /tmp/wf_panel_output, pero puede apuntar a cualquier ruta accesible para el servicio Odoo.
  • Garantizar permisos de lectura y escritura sobre dicha carpeta para el usuario que ejecuta Odoo.

Dependencias Python

Instale las siguientes librerías en el entorno de Odoo:

  • pdfplumber
  • PyPDF2
  • numpy
  • trimesh
  • lxml
  • pillow
  • pytesseract
  • opencv-python
  • svgwrite

Dependencias de sistema

  • tesseract-ocr
  • inkscape (para generar SVG y realizar conversiones)

Resultados obtenidos

  • Paneles woodframe con dimensiones, capas y referencias de materiales.
  • Listas de materiales y rutas de fabricación alineadas con cada panel.
  • Órdenes de fabricación generadas automáticamente listas para planificar.
  • Archivos de reportes y depuración guardados en la carpeta configurada.
